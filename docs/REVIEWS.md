# Reviews

Project: **OpenAI-compatible YOLO11 COCO Detection API**

This file stores strategic review summaries for PRs, audits, and release decisions.

---

## Review doctrine

A PR report is a claim, not proof.

Review should check:

- goal match;
- diff scope;
- tests;
- documentation;
- security;
- compatibility;
- release claims;
- remaining risks.
